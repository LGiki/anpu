# Anpu
一个收集焦安溥歌词、演出歌单、Talking的网站：[https://anpu.me](https://anpu.me)

## Features

- 网站内容
  - 歌词
  - 专辑
  - 演出歌单
  - Talking
  - 演出统计
- 支持 PWA，可在离线状态下查看全部歌词
- 支持 [llms.txt](https://llmstxt.org/)：[https://anpu.me/llms.txt](https://anpu.me/llms.txt)
